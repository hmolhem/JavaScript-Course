REST API

REST is stand for Representational State Transfer

    http request include the four following form:
  • GET: request information from server (READ)
  • POST: send and save information to server (CREATE)
  • PUT: update information on server (UPDATE)
  • DELETE: delete information on server (DELETE)

API endpoint is url addrres for sending request

  • GET: http://example.com/api/posts // Get all posts
  • GET: http://example.com/api/posts/1 // Get single post
  • POST: http://example.com/api/posts // Add post
  • PUT: http://example.com/api/posts/1 // Update post
  • DELETE: http://example.com/api/posts/1 // Delete post

Status Code

when you send a request then server send a status code to you

  • 1xx: information
  • 2xx: success
  • 3xx: redirection
  • 4xx: client error
  • 5xx: server error
